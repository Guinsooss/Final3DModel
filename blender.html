Blender
Blender is a free to use open sourse 3D modeling computer graphics software that allows the user to create aminated films, video editing, art peices, 3D printed models, animated scenes, modeling and video games. Inside blender there is a range of tools the user can use, for example 3D Modeling, wrapping and unwrapping images, texturing, sculpting, animating, camera tracking, rendering, motion graphics and even has an intergrated game engine.
