Progress Shots in order
To create my object I needed to use the mesh tool which lets me create many different shapes, for example a cube, sphere, circle, cone and a plane (flat surface). I also needed to use the scale tool which lets me change the size of the whole shape, edit mode which allows me to select a face of an object, a vertex of an object and an edge of an object. I also had to use the extrude tool which allows me to extend one part of an object, for example using the edit tool I can select the top face of a cube and then use the extrude tool to extend only that selected face. I also needed the material tool to change the look of the object. Inside the material tool I used the diffuse tool which allows me to change the colour of the object using the colour picker supplied inside the program, inside the material tool I used the texture tool to change the pattern on the object, this adds detail to the object.
